Sonic Forage Infinite Radio

A public browser prototype for endless generated music: the current chunk plays, the next prompt cooks in the queue, then the deck crossfades into it.

Prompt next chunk

Local mode uses deterministic Web Audio synthesis from the prompt so the demo works for everybody. Modal mode expects an API that returns { audio_url } or { audio_base64 }.

Modal backend

  • Default public mode: no secret, no surprise GPU spend.
  • Operator mode: paste your Modal endpoint/key and queue real chunks.
  • Playback: chunks are decoded in browser and crossfaded by Web Audio.
